## 程式介紹
Black Jack 21 點是一款撲克牌遊戲，玩家的目標是使手中的牌點數之和盡量接近或等於21點，但不能超過。玩家與莊家（電腦）進行比較，誰的牌點數更接近21點就贏得本局。
